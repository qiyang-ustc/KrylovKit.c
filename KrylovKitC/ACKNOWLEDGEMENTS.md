# Acknowledgements

KrylovKit.c exists because KrylovKit.jl provides a precise, well-designed
reference for Krylov algorithms in Julia. We thank Jutho Haegeman and all
KrylovKit.jl contributors for that work.

This package uses KrylovKit.jl as the semantic reference in tests and benchmarks.
Any performance comparison is a backend comparison for the listed workload,
hardware, tolerance, and commit. It should not be read as a criticism of
KrylovKit.jl, whose design goals include generic Julia semantics that are broader
than this native backend.
